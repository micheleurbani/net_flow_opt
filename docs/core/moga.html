Multi-objective genetic algorithm¶

generate_individual(i=None)¶

The method generates a feasible grouping structure. The following two constraints are satisfied:

\[ \begin{align}\begin{aligned}|G| \le R \qquad \forall G \in grouping\_structure\\\min_{c \in G} \{t_c + d_c\} > \max_{c \in G} \{t_c\} \qquad \forall G \in grouping\_structure\end{aligned}\end{align} \]

where \(G\) identifies a set of components also called group, which ensure that the optimality of nestde intervals holds.

Returns

a numpy array encoding the grouping structure.

Return type

numpy array

hypervolume_indicator(n_sample, lf_max, lf_min, ic_max, ic_min)¶

The method calculates the hyper-volume (HV) indicator of each generation and returns the series of points required to plot it. It is required to provide \(LF_{max}\), \(LF_{min}\), \(IC_{max}\), and \(IC_{min}\) so that several HV curves can be compared.

Params int n_sample

the number of sample points used to evaluate the

HV indicator. :params float lf_min: the lower bound for \(LF\). :params float lf_max: the upper bound for \(LF\). :params float ic_min: the lower bound for \(IC\). :params float ic_max: the upper bound for \(IC\).

Returns

the HV indicator value for each generation.

Return type

list of float.
