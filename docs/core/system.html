The system module¶

The system module contains the data structures required to model system components.

class core.system.Component(cc, cp, alpha, beta, capacity)¶

The object contains the parameters relevant to the the component and to its failure distribution.

Parameters

  • cc (float) – the cost of a corrective intervention.

  • cp (float) – the cost of a preventive intervention.

  • alpha (float) – the scale factor of the Weibull 2-paramters |br|

distribution. :param float beta: the shape factor of the Weibull 2-paramters distribution. :param float capacity: the amount of material processed per unit of time.

info()¶
class core.system.System(structure, resources, components)¶

The object contains the information about the structure of the system.

Parameters

  • structure (object) – a networkx.DiGraph object.

  • resources (int) – the number of available repairmen staff.

static from_node_to_edge_capacity(structure)¶

The function returns the edge capacited version of a node capacited network.

Parameters

system_structure (object) – the networkx.DiGraph object that

describes the original structure of the network.

Returns

a networkx.DiGraph object with twinned nodes and edge

capacitiees. The capacity of the old edges is set to infinite. :rtype: object
